# Building an interpreter in golang

Interpreter built in Golang for the Monkey programming language inspired by the book "Writing an Interpreter in GO" by Thorsten Ball

Monkey Programming language has the following: 

```
• C-like syntax
• variable bindings
• integers and booleans
• arithmetic expressions
• built-in functions
• first-class and higher-order functions
• closures
• a string data structure
• an array data structure
• a hash data structure
```
